PROJECT OVERVIEW
----------------
NovaAudioVideoConverter is a comprehensive, open-source application for
Linux that provides professional audio and video conversion capabilities
with AI-powered enhancement features.

MAIN FEATURES
-------------
✓ 180+ Format Support (81 implemented, extensible)
✓ 200+ Device Profiles (170 implemented, extensible)
✓ DVD Ripping with Protection Removal
✓ Video Editing (Cut, Rotate, Crop, Merge)
✓ AI Audio Enhancement (Denoise, Repair, Normalize)
✓ AI Video Enhancement (Denoise, Sharpen, Stabilize, Upscale)
✓ Subtitle Search & Management
✓ Hardware Acceleration Support
✓ PyQt5 GUI with 5 Main Tabs
✓ Batch Processing Ready

INSTALLATION
------------
1. Clone repository:
   git clone https://github.com/linuxiano85/NovaAudioVideoConverter.git
   cd NovaAudioVideoConverter

2. Run installation script:
   chmod +x install.sh
   ./install.sh

3. Launch application:
   python3 main.py

QUICK START
-----------
GUI Mode:
  python3 main.py

Examples:
  python3 examples.py

Command Line:
  from src.core.converter import Converter
  converter = Converter()
  converter.convert('input.avi', 'output.mp4')

1. CONVERSION
   - 180+ formats (video and audio)
   - Hardware acceleration
   - Custom profiles
   - Batch processing
   - Lossless merging

2. DVD RIPPING
   - Physical DVD extraction
   - ISO file conversion
   - Protection removal (CSS/AACS)
   - Multi-title support
   - Subtitle extraction

3. VIDEO EDITING
   - Precise cutting/trimming
   - Rotation (90/180/270°)
   - Custom cropping
   - File merging
   - Filter application

4. AI AUDIO ENHANCEMENT
   - Intelligent noise removal
   - Voice enhancement
   - Click/pop removal
   - Volume normalization
   - Dynamic compression
   - Damaged audio repair

5. AI VIDEO ENHANCEMENT
   - Noise reduction

   - Stabilization
   - AI upscaling (2x/3x/4x)

   - Color enhancement
   - Damaged video repair
